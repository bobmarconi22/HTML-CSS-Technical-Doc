Introduction

HTML is the logic end of front end web development, it stands for HyperText Markup Language. HTML is a great way to provide information or access to a product, and it is the bare-bones of every website. HTML makes up anything you've ever visited on the worldwide web,

CSS is the design side of things. CSS stands for Cascading Style Sheets. This is where web pages get their colors, borders, boxes, interactivity, or any other type of visual boost. the following is an example of an opening and closing tag for the HTML element 'body'.

< body > **content** < /body >

    • CSS

    CSS or "Cascading Style Sheets" is the design of web design. it is everything you see in a website that has color, borders, and most spacing/formatting. Almost everything you see on this page was designed using CSS, the background color, the orange text for the code in each section, and the navagation bar to the left.

    Hovering is just one example of what CSS can do to a document. Notice how when you hover your cursor over the links in the navigation bar how the text gets bold? Pretty cool right? Well it can do many, many other things too! It can change the color of the text, the background, give the border shadow, the possibilities are endless! The syntax for a CSS document is as follows

    selector{ specifications }

    Some CSS selectors and elements include:

  • .id-name
  • #class-name
  • element-name
  • background-color
  • display
  • border
  • @media
    • Front End Development

    Front end development is the combination of both HTML and CSS to create UI's (User Interfaces) for consumers to use and operate easily. UI's are important for any industry, wether supplying a service, selling a product, or giving users access to something entirely different, the UI is where consumers will be able to digital access the goods or your business information.

    Potentially the most important thing about creating UI's is accessability. Making a clear, concise, neat, easy to use website or interface is key to a successful site. Think about a website you've visited that has been easy to navigate, what comes to mind? Apple? Samsung? Google?
